# QrMaker
**QrGen** _is detailed Qr Code generator_

## Installation
Install "requirements.txt" for Libraries
```
pip install -r requirements.txt
```

## Useage
```
python3 QrMaker.py
```

## Options
```
1)URL
2)TEXT
3)PHONE
4)WIFI
```
