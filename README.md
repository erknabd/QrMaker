# QrMaker
**QrMaker** _is detailed Qr Code generator_

## Installation
Install "requirements.txt" for Libraries
```
git clone https://github.com/shexer/QrMaker
pip install -r requirements.txt
```

## Useage
```
python3 QrMaker.py
```

## Options
```
1)URL
2)TEXT
3)PHONE
4)WIFI
```
